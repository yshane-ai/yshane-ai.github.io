Pythonзҡ„еӯ—е…ёdictеҺҹзҗҶ

2021-01-13
Python

Table of Contents

дёҖ. д»Җд№ҲжҳҜеӯ—е…ёпјҹ

еӯ—е…ёжҳҜдёҖзі»еҲ—з”ұй”®пјҲkeyпјүе’ҢеҖјпјҲvalueпјүй…ҚеҜ№з»„жҲҗзҡ„е…ғзҙ зҡ„йӣҶеҗҲгҖӮеӯ—е…ёжҳҜдёҖдёӘеҸҜеҸҳе®№еҷЁжЁЎеһӢпјҢеҸҜд»ҘеӯҳеӮЁд»»ж„Ҹзұ»еһӢеҜ№иұЎгҖӮеӯ—е…ёе®һзҺ°дёҺе“ҲеёҢз®—жі•еҜҶдёҚеҸҜеҲҶпјҲдёҚеҗҢзҡ„PythonзүҲжң¬пјҢз®—жі•дјҡдёҚеҗҢпјүпјҢдёҚдәҶи§Је“ҲеёҢз®—жі•зҡ„з«ҘйһӢеҸҜд»Ҙе…ҲеҺ»дәҶи§Јзӣёе…ізҹҘиҜҶгҖӮ

дәҢ. еӯ—е…ёжҳҜеҗҰжҳҜжңүеәҸзҡ„пјҹ

еңЁPython3.6д№ӢеүҚпјҢеӯ—е…ёжҳҜж— еәҸзҡ„пјҢдҪҶжҳҜPython3.7+пјҢеӯ—е…ёжҳҜжңүеәҸзҡ„гҖӮеңЁ3.6дёӯпјҢеӯ—е…ёжңүеәҸжҳҜдёҖдёӘimplementation detailпјҢеңЁ3.7жүҚжӯЈејҸжҲҗдёәиҜӯиЁҖзү№жҖ§пјҢеӣ жӯӨ3.6дёӯж— жі•зЎ®дҝқ100%жңүеәҸгҖӮ

дёү. еӯ—е…ёзҡ„жҹҘиҜўгҖҒж·»еҠ гҖҒеҲ йҷӨзҡ„ж—¶й—ҙеӨҚжқӮеәҰпјҹ

еӯ—е…ёзҡ„жҹҘиҜўгҖҒж·»еҠ гҖҒеҲ йҷӨзҡ„е№іеқҮж—¶й—ҙеӨҚжқӮеәҰйғҪжҳҜO(1)пјҲдёәд»Җд№ҲжҳҜе№іеқҮж—¶й—ҙеӨҚжқӮеәҰпјҢж–Үз« еҗҺйқўдјҡи®Іи§ЈеҲ°пјүпјҢзӣёжҜ”еҲ—иЎЁдёҺе…ғзҘ–пјҢжҖ§иғҪжӣҙдјҳгҖӮ

еӣӣ. еӯ—е…ёзҡ„е®һзҺ°еҺҹзҗҶпјҹ

1. йҰ–е…ҲиҜҙиҜҙPython3.6д№ӢеүҚзҡ„ж— еәҸеӯ—е…ё

еӯ—е…ёеә•еұӮжҳҜз»ҙжҠӨдёҖеј е“ҲеёҢиЎЁпјҲи§ҒдёӢеӣҫпјүпјҢжҲ‘д»¬еҸҜд»ҘжҠҠе“ҲеёҢиЎЁзңӢжҲҗдёҖдёӘеҲ—иЎЁпјҢе“ҲеёҢиЎЁдёӯзҡ„жҜҸдёҖдёӘе…ғзҙ еҸҲеӯҳеӮЁдәҶе“ҲеёҢеҖјпјҲhashпјүгҖҒй”®пјҲkeyпјүгҖҒеҖјпјҲvalueпјү3дёӘе…ғзҙ гҖӮпјҲPython3.6д№ӢеүҚпјү

enteies = [
    ['--', '--', '--'],
    [hash, key, value],
    ['--', '--', '--'],
    ['--', '--', '--'],
    [hash, key, value],
]

з”ұдёҠеҸҜд»Ҙи§Ғе“ҲеёҢиЎЁзҡ„еӯҳеӮЁз»“жһ„пјҢжҲ‘д»¬д№ҹеҸҜд»ҘзңӢеҮәпјҢе…ғзҙ д№Ӣй—ҙжңүдёҖдәӣз©әе…ғзҙ пјҢжҲ‘д»¬йҖҡиҝҮеўһеҠ дёҖдёӘе…ғзҙ жқҘи®Іи§Је…·дҪ“е®һзҺ°гҖӮ

  • и®Ўз®—keyзҡ„hashеҖјгҖҗhash(key)гҖ‘пјҢеҶҚе’ҢmaskеҒҡдёҺж“ҚдҪңгҖҗmask=еӯ—е…ёжңҖе°Ҹй•ҝеәҰпјҲDictMinSizeпјү - 1гҖ‘пјҢиҝҗз®—еҗҺдјҡеҫ—еҲ°дёҖдёӘж•°еӯ—гҖҗindexгҖ‘пјҢиҝҷдёӘindexе°ұжҳҜиҰҒжҸ’е…Ҙзҡ„enteiesе“ҲеёҢиЎЁдёӯзҡ„дёӢж ҮдҪҚзҪ®
  • иӢҘindexдёӢж ҮдҪҚзҪ®е·Із»Ҹиў«еҚ з”ЁпјҢеҲҷдјҡеҲӨж–ӯenteiesзҡ„keyжҳҜеҗҰдёҺиҰҒжҸ’е…Ҙзҡ„keyжҳҜеҗҰзӣёзӯү
  • еҰӮжһңkeyзӣёзӯүе°ұиЎЁзӨәkeyе·ІеӯҳеңЁпјҢеҲҷжӣҙж–°valueеҖј
  • еҰӮжһңkeyдёҚзӣёзӯүпјҢе°ұиЎЁзӨәhashеҶІзӘҒпјҢеҲҷдјҡз»§з»ӯеҗ‘дёӢеҜ»жүҫз©әдҪҚзҪ®пјҢдёҖзӣҙеҲ°жүҫеҲ°еү©дҪҷз©әдҪҚдёәжӯўгҖӮ

2. Python3.7+еҗҺзҡ„ж–°зҡ„е®һзҺ°ж–№ејҸ

иҖҒеӯ—е…ёдҪҝз”ЁдёҖеј hashпјҢиҖҢж–°еӯ—е…ёиҝҳдҪҝз”ЁдәҶдёҖеј IndicesиЎЁжқҘиҫ…еҠ©гҖӮдёӢжқҘеҲ—еҮәж–°зҡ„з»“жһ„пјҡ

indices = [None, None, index, None, index, None, index]
enteies = [
    [hash0, key0, value0],
    [hash1, key1, value1],
    [hash2, key2, value2]
]

дёӢйқўдёәе…·дҪ“зҡ„е®һзҺ°иҝҮзЁӢпјҡ

  • и®Ўз®—keyзҡ„hashеҖјгҖҗhash(key)гҖ‘пјҢеҶҚе’ҢmaskеҒҡдёҺж“ҚдҪңгҖҗmask=еӯ—е…ёжңҖе°Ҹй•ҝеәҰпјҲIndicesDictMinSizeпјү - 1гҖ‘пјҢиҝҗз®—еҗҺдјҡеҫ—еҲ°дёҖдёӘж•°еӯ—гҖҗindexгҖ‘пјҢиҝҷдёӘindexе°ұжҳҜиҰҒжҸ’е…Ҙзҡ„indicesзҡ„дёӢж ҮдҪҚзҪ®пјҲжіЁпјҡе…·дҪ“з®—жі•дёҺPythonзүҲжң¬зӣёе…іпјҢе№¶дёҚдёҖе®ҡдёҖж ·пјү
  • еҫ—еҲ°indexеҗҺпјҢдјҡжүҫеҲ°indicesзҡ„дҪҚзҪ®пјҢдҪҶжҳҜжӯӨдҪҚзҪ®дёҚжҳҜеӯҳзҡ„hashеҖјпјҢиҖҢжҳҜеӯҳзҡ„len(enteies)пјҢиЎЁзӨәиҜҘеҖјеңЁenteiesдёӯзҡ„дҪҚзҪ®
  • еҰӮжһңеҮәзҺ°hashеҶІзӘҒпјҢеҲҷеӨ„зҗҶж–№ејҸдёҺиҖҒеӯ—е…ёеӨ„зҗҶж–№ејҸзұ»дјј

дә”. ж—¶й—ҙеӨҚжқӮеәҰиҜҙжҳҺ

жҲ‘д»¬еңЁдёҠйқўжҸҗеҲ°дәҶпјҢеӯ—е…ёзҡ„е№іеқҮж—¶й—ҙеӨҚжқӮеәҰжҳҜO(1)пјҢеӣ дёәеӯ—е…ёжҳҜйҖҡиҝҮе“ҲеёҢз®—жі•жқҘе®һзҺ°зҡ„пјҢе“ҲеёҢз®—жі•дёҚеҸҜйҒҝе…Қзҡ„й—®йўҳе°ұжҳҜhashеҶІзӘҒпјҢPythonеӯ—е…ёеҸ‘з”ҹе“ҲеёҢеҶІзӘҒж—¶пјҢдјҡеҗ‘дёӢеҜ»жүҫз©әдҪҷдҪҚзҪ®пјҢзӣҙеҲ°жүҫеҲ°дҪҚзҪ®гҖӮеҰӮжһңеңЁи®Ўз®—keyзҡ„hashеҖјж—¶пјҢеҰӮжһңдёҖзӣҙжүҫдёҚеҲ°з©әдҪҷдҪҚзҪ®пјҢеҲҷеӯ—е…ёзҡ„ж—¶й—ҙеӨҚжқӮеәҰе°ұеҸҳжҲҗдәҶO(n)дәҶпјҢжүҖд»ҘPythonзҡ„е“ҲеёҢз®—жі•е°ұжҳҫеҫ—йқһеёёйҮҚиҰҒдәҶгҖӮPythonеӯ—е…ёзҡ„е“ҲеёҢз®—жі•пјҢдјҡе°ҪйҮҸдҝқиҜҒе“ҲеёҢеҖји®Ўз®—еҮәзҡ„indexжҳҜе№іеқҮеҲҶеёғдё”жҜҸдёҖдёӘеҖјд№Ӣй—ҙжңүеү©дҪҷдҪҚзҪ®гҖӮ

е…ӯ. еӯ—е…ёзҡ„keyиғҪдҪҝз”Ёд»Җд№ҲеҖјпјҹ

Pythonеӯ—е…ёзҡ„keyеҸҜд»ҘдҪҝз”Ёеӯ—з¬ҰдёІпјҲstrпјүпјҢж•ҙеһӢпјҲintпјүпјҢе…ғзҘ–пјҲtupleпјүзӯүгҖӮжҲ‘д»¬е·Із»ҸзҹҘйҒ“пјҢеӯ—е…ёжҳҜйҖҡиҝҮе“ҲеёҢз®—жі•жқҘи®Ўз®—keyзҡ„еҖјпјҢжүҖд»Ҙkeyеҝ…йЎ»дёәеҸҜе“ҲеёҢзҡ„пјҢlistдёҚиғҪдҪңдёәеӯ—е…ёзҡ„keyпјҢеӣ дёәlistжҳҜеҸҜеҸҳзҡ„еҸҠдёҚеҸҜе“ҲеёҢзҡ„еҜ№иұЎпјҢжүҖд»ҘдёҚиғҪдҪңдёәеӯ—е…ёзҡ„keyгҖӮ
